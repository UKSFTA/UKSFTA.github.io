# Security Policy

## Operational Security (OPSEC)

This project is dedicated to the representation of the United Kingdom Special Forces within a milsim environment. It is a critical mandate that all community members adhere to strict OPSEC protocols:

- Never include real-world personnel names, faces, or sensitive personal identifiers.
- Never include specific coordinates or sensitive data related to real-world operations or personnel.
- All community activities are simulated and for entertainment purposes only.

## Reporting a Vulnerability

If you discover a security vulnerability within this theme or its components, please contact the Taskforce Alpha command team via Discord.

## Automated Scanning

This repository utilizes automated tools to identify potential vulnerabilities. Pull requests that do not pass security audits will be reviewed by community leads.
