# Security Policy

## Supported Versions

Only the latest version of the bot is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue. Instead, report it to the UKSF Directorate or the maintainers directly.

All security reports will be acknowledged within 48 hours, and a fix will be prepared as soon as possible.
