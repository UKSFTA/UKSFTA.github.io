# Security Policy

## Operational Security (OPSEC)

This project is dedicated to the representation of the United Kingdom Special Forces. It is a critical mandate that all contributors and maintainers adhere to strict OPSEC protocols:

- Never include real-world personnel names, faces, or identifiers.
- Never include specific coordinates or sensitive operational locations.
- Ensure all imagery and data are either simulated or approved for public release.

## Reporting a Vulnerability

If you discover a security vulnerability within this theme or its components, please do not use the public issue tracker. Instead, follow the standard MOD disclosure protocols or contact the project maintainers via encrypted channels.

## Automated Scanning

This repository utilizes CodeQL and other automated tools to identify potential vulnerabilities. PRs that do not pass security audits will be blocked.
